# 💕 Heartbound Adventures - Romantic RPG Design Document

> A cute, cozy co-op RPG inspired by Minecraft exploration + Sky: Children of the Light's emotional connection + Animal Crossing's gentle gameplay

## 🎨 Core Concept

**Heartbound Adventures** is a wholesome multiplayer RPG where you and your partner explore a magical world together, building memories, discovering treasures, and strengthening your bond through cooperative gameplay.

### Design Pillars

1. **Cozy & Non-Violent** - No combat, only discovery and cooperation
2. **Emotional Connection** - Gameplay centered around togetherness
3. **Creative Expression** - Build, decorate, and customize together
4. **Meaningful Progress** - Every action strengthens your bond
5. **Beautiful & Relaxing** - Soft colors, gentle music, peaceful atmosphere

---

## 🌍 World Design

### The Realm of Luminara

A magical floating world made of interconnected islands, each with unique biomes:

#### Starter Islands
1. **Meadow Haven** 🌸 - Tutorial island with flowers and butterflies
2. **Whispering Woods** 🌲 - Peaceful forest with hidden secrets
3. **Starlight Beach** ⭐ - Bioluminescent shore at eternal twilight

#### Advanced Islands
4. **Crystal Caverns** 💎 - Underground wonder with glowing crystals
5. **Cloud Gardens** ☁️ - Sky-high floating gardens
6. **Moonlit Lake** 🌙 - Serene water world with lily pads
7. **Aurora Peak** 🏔️ - Mountain top with dancing lights
8. **Enchanted Village** 🏘️ - NPCs, shops, and community hub

### Navigation
- **Spirit Bridges** - Glowing paths that appear when both players hold hands
- **Teleport Stones** - Unlockable fast travel between islands
- **Flying Mounts** - Cute creatures you befriend and ride together

## 👥 Character System

### Character Creation (Cute & Cozy)

**Avatar Style**: Chibi/kawaii characters with:
- Large expressive eyes
- Soft rounded shapes
- Pastel color palettes
- Customizable outfits and accessories

**Customization Options**:
- **Body**: Height (smol, medium, tall), Body type (soft variations)
- **Face**: 12 eye styles, 8 mouth styles, blush variations
- **Hair**: 20+ styles (short, long, curly, straight, braids)
- **Colors**: Pastels, vibrant, natural tones
- **Outfits**: Casual, fantasy, seasonal, matching couple sets
- **Accessories**: Hats, glasses, wings, tails, flowers

### No Traditional Stats

Instead of combat stats, characters have:

- **Bond Level** (1-100) - Shared between both players
- **Exploration Progress** - Islands discovered
- **Collection Completion** - Items found together
- **Home Level** - Your shared space upgrades

## 🎮 Gameplay Mechanics

### Core Activities

#### 1. **Exploration** 🗺️
- Discover new islands together
- Find hidden caves and secret areas
- Collect memory fragments (story pieces)
- Take photos at scenic viewpoints

#### 2. **Gathering & Crafting** 🌼
- Pick flowers, fruits, and mushrooms
- Mine crystals and gems
- Fish in magical ponds
- Craft furniture, decorations, and gifts
- No inventory limits - stress-free collection

#### 3. **Home Building** 🏡
- Shared home that grows with your bond
- Place furniture and decorations together
- Unlock new rooms and areas
- Garden where you plant flowers together
- Pet sanctuary for creatures you befriend

#### 4. **Bonding Activities** 💑
- **Hold Hands** - Reveals hidden paths, restores energy
- **Emotes** - Wave, hug, dance, high-five (with animations)
- **Gift Exchange** - Craft items for each other
- **Photo Mode** - Take selfies and screenshots together
- **Stargazing** - Sit together and watch the sky
- **Music Making** - Play instruments in harmony

#### 5. **Creature Befriending** 🦋
- No fighting - only befriending
- Feed creatures their favorite foods
- Play mini-games to earn trust
- Ride/fly together on larger creatures
- Collect creature "friends" who visit your home

#### 6. **Puzzle Solving** 🧩
- Cooperative puzzles requiring both players
- Light beam puzzles (redirect light together)
- Harmony puzzles (play musical notes in sync)
- Balance puzzles (stand on pressure plates)
- Memory puzzles (match patterns together)

## 🛠️ Technical Implementation

### Technology Stack

**Frontend**:
- React + TypeScript (existing setup)
- Three.js or Babylon.js for 3D rendering
- React Three Fiber for React integration
- Zustand for state management
- Framer Motion for UI animations

**Backend**:
- Firebase Realtime Database (player positions, real-time sync)
- Supabase PostgreSQL (persistent data: progress, inventory, home)
- Firebase Storage (user photos, custom content)

**Database Schema Extensions**:

```sql
-- RPG Character Data
CREATE TABLE rpg_characters (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_email TEXT NOT NULL,
  character_name TEXT NOT NULL,
  appearance JSONB, -- body, face, hair, colors
  outfit JSONB,     -- current outfit and accessories
  position JSONB,   -- current world position
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Shared Couple Progress
CREATE TABLE couple_progress (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  player_1_email TEXT NOT NULL,
  player_2_email TEXT NOT NULL,
  bond_level INTEGER DEFAULT 1,
  bond_exp INTEGER DEFAULT 0,
  islands_unlocked TEXT[] DEFAULT ARRAY['meadow_haven'],
  achievements JSONB DEFAULT '[]',
  total_play_time INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Home/Island Data
CREATE TABLE couple_home (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  couple_id UUID REFERENCES couple_progress(id),
  home_level INTEGER DEFAULT 1,
  furniture JSONB DEFAULT '[]', -- placed furniture items
  unlocked_rooms TEXT[] DEFAULT ARRAY['living_room'],
  garden_plants JSONB DEFAULT '[]',
  pets JSONB DEFAULT '[]',
  decorations JSONB DEFAULT '{}',
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Inventory
CREATE TABLE rpg_inventory (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  couple_id UUID REFERENCES couple_progress(id),
  items JSONB DEFAULT '{}', -- {item_id: quantity}
  materials JSONB DEFAULT '{}',
  collectibles JSONB DEFAULT '[]',
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Memory Book
CREATE TABLE memory_book (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  couple_id UUID REFERENCES couple_progress(id),
  entry_type TEXT NOT NULL, -- 'photo', 'milestone', 'first_visit'
  entry_data JSONB NOT NULL,
  entry_date TIMESTAMP DEFAULT NOW()
);

-- Daily Challenges
CREATE TABLE daily_challenges (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  challenge_date DATE NOT NULL,
  challenges JSONB NOT NULL, -- array of 5 daily challenges
  created_at TIMESTAMP DEFAULT NOW()
);

-- Challenge Completion
CREATE TABLE challenge_completion (
  couple_id UUID REFERENCES couple_progress(id),
  challenge_date DATE NOT NULL,
  completed_challenges TEXT[],
  rewards_claimed BOOLEAN DEFAULT FALSE,
  PRIMARY KEY (couple_id, challenge_date)
);
```

**Keyboard/Mouse**:
- WASD - Move
- Mouse - Camera
- E - Interact
- Space - Jump/Fly

- F - Hold hands (when near partner)

- Tab - Open menu
- P - Photo mode

## 💡 Why This Works for a Couple's Game

### 1. **No Competitive Elements**
- Pure cooperation, no PvP
- Shared progress means you win together
- No pressure or stress

### 2. **Flexible Play Sessions**
- No time pressure
- Can play 15 minutes or 3 hours
- Progress saves automatically
- Pick up where you left off

### 3. **Creates Shared Memories**
- Photo mode captures moments
- Memory book preserves milestones
- Inside jokes from gameplay
- "Remember when we found that secret cave?"

### 4. **Low Barrier to Entry**
- Simple controls
- No complex mechanics to learn
- Accessible for non-gamers
- Beautiful and inviting

### 5. **Long-Term Engagement**
- 100 bond levels to reach
- Seasonal events keep it fresh
- Daily challenges provide routine
- New islands to discover

**Heartbound Adventures** combines the exploration of Minecraft, the emotional connection of Sky: Children of the Light, and the cozy customization of Animal Crossing into a unique experience designed specifically for couples.

The game focuses on:
- ✨ Creating magical moments together

It's not about winning or losing—it's about the journey you take together.
